- 经济奔溃，失业无业者越多，娱乐业可能兴起（参考知乎日本，抚州麻将）
- 30岁以前最重要的是“成长”而非“财富积累”
- M0,M1,M2,M3流动性下降
 + M0：现金
 + M1：M0+活期存款 （M1增长过快，消费和终端市场活跃）
 + M2：M1+定期存款  （M2增长过快，投资和中间市场活跃）
 + M3：M2+具有高流动性的证券和其它资产
 + M1过高M2过低，表明需求强劲、投资不足，存在通货膨胀风险
 + M2过高而M1过低，表明投资过热、需求不旺，存在资产泡沫风险
